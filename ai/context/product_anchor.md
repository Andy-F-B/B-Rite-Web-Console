# PRODUCT ANCHOR

App: B-Rite Console

Users can:
- Create account and sign in
- Write b-rite scripts with auto-formatting, Insert (saved scripts), syntax validation, line numbers
- Choose pre-loaded auto prompts; web-sdk: feature library info/commands, createFeaturePack in Ctrl+Alt+D
- Download template, Web SDK (zip), Documentation PDFs; install plugins
- Save scripts to account (folders, archive); Post to Public Library from Saved
- Browse Public Library — share and discover code snippets; thumbs up, copy when allowed

Primary entities: profiles, scripts, folders, saved_items, plugins, public_posts, post_reactions
